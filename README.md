# Find My Coop

## Description:  

Find my coop est un site internet hybride (forum / outil gestion de projet), qui vise à pérenniser les projets perso voués à l’échec, notamment à cause des compétences à mettre en œuvre pour la réussite .



## Logiciels utilisés:  

- NetBeans 8.2
- Zend 2.5
- PHP 5
- mySQL 5.7
- phpmyadmin 4.5.4.1
